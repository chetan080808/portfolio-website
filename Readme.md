## üéØ Overview

This portfolio website was built to showcase my journey as a **Cloud Support Engineer** at Hexaware Technologies, highlighting my expertise in AWS, DevOps, and cloud infrastructure management. The project emphasizes modern design, smooth animations, and optimal performance while maintaining a completely **free hosting solution**.

### üé® Design Philosophy
- Modern & Professional - Glassmorphism UI with smooth animations
- Performance-First - Optimized for fast load times and SEO
- Mobile-Responsive - Seamless experience across all devices
- Accessibility - WCAG 2.1 compliant design patterns

## ‚ú® Features

### Interactive UI Components
- **Animated Navigation Bar** - Floating particle effects, active section detection, glassmorphism design, responsive hamburger menu

## üåê Deployment

### Why Vercel?

Chose Vercel as the hosting platform due to:

‚úÖ **Zero Cost** - Free tier perfect for personal portfolios  
‚úÖ **Automatic Deployments** - Push to GitHub equals instant deployment  
‚úÖ **Global CDN** - Lightning-fast content delivery worldwide  
‚úÖ **SSL Certificate** - Free HTTPS out of the box  
‚úÖ **Zero Configuration** - Works with React automatically  
‚úÖ **Serverless** - Scales automatically without management  
‚úÖ **Custom Domains** - Free subdomain plus custom domain support  

### Deployment Process

1. Push code to GitHub
git add .
git commit -m "Deploy to production"
git push origin main

text

2. Vercel auto-deploys - Detects changes in main branch, runs build process, deploys to global CDN, provides instant preview URL

3. Live in 2-3 minutes

### Alternative Platforms Considered

**Netlify** - Free but Vercel has better React optimization

**GitHub Pages** - Free but requires manual builds, no serverless

**AWS S3 + CloudFront** - Costs $5-10 per month, more configuration needed

**Heroku** - Free tier ending, limited free options

**Firebase Hosting** - Free but Vercel simpler for React apps

**Winner: Vercel** - Best balance of performance, features, and zero cost
